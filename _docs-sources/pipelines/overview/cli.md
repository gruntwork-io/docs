# Pipelines CLI

Generally speaking, most users of Pipelines should not need to install the Pipelines CLI on their local workstations.

The binary is designed to be run in a CI/CD environment, and Pipelines GitHub Actions workflows are provided to ensure that the binary is available, and used correctly in the context of a Pipelines run.

There are particular use-cases where it can be beneficial to install the Pipelines CLI on your local workstation, however.
